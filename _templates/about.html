Media Database

A local movie database for your personal collection

Media Database creates a local movie database containing information related to your movie collection. Webpages are statically generated and uploaded to the _output/ directory. Rerun the script every time a new file is added to the directory.

Search Filter Usage

Multiple criteria: Genres, Cast, and Language fields support comma-separated queries

"Sci-Fi, Action" → Returns titles that are both Sci-Fi and Action

Negative queries: Use ! to exclude criteria

"Sci-Fi, Action, !Drama" → Sci-Fi and Action titles, excluding Drama

Partial matching: Searches match from the start of the string

"Po" → Returns Portuguese, Polish, etc.
