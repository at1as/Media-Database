IMDB-Scrape creates a local movie database containing information related to your movie collection. Webpages are staticly generated and uploaded to the _output/ directory. Rerun the script every time a new file is added to the directory

Search Filter Usage

Genres, Cast, and Language fields support multiple comma-separated queries:

-> "Sci-Fi, Action"
-> Returns titles that are both Sci-Fi and Action

As well as negative queries:

-> "Sci-Fi, Action, !Drama"
-> Returns titles that are marked as both Sci-Fi and Action while not being marked as Drama

Search queries do a match on the start of the string:

-> "Po"
-> Returns titles marked as Portuguese, Polish, etc
